# Description
Findit is a program designed to search word files for search/sniff words. 

# Compile
c++ -std=c++17 -o main *.cpp

# Run
If you want to look for the word CSCI in Folder1 then print out to found.txt
main --dir /Folder1 CSCI --verbose
